# @tuvren/core

## 1.0.0

### Minor Changes

- First public registry release of the Tuvren SDK (Epic BL, ADR-054/056/057): the frozen stable core (`@tuvren/core`, `@tuvren/sdk`), the published leaf packages, and the internal engine packages (`@tuvren/runtime`, `@tuvren/kernel-protocol`, `@tuvren/kernel-runtime`, `@tuvren/provider-api`, `@tuvren/telemetry-semconv`) published for dependency resolution only. Versions move in lockstep via the fixed `@tuvren/*` group.
