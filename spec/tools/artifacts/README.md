# Tool Contracts Reviewed Artifacts

Epic R activates this directory as the reviewed-artifact home for
machine-readable outputs owned by the `tuvren.shared.core` packet exposed
through the `spec/tools` port (regenerated via `tools-spec:codegen`).

- Checked-in files here are reviewed outputs, not primary semantic sources.
- Artifact additions must remain aligned with `docs/` and `.constitution/`.
